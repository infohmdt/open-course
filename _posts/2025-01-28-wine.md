## Wine
Wine adalah sebuah program yang memungkinkan pengguna untuk menjalankan aplikasi dan permainan Windows di lingkungan sistem operasi Linux, termasuk Kali Linux. Wine bukanlah emulator atau mesin virtual; melainkan, ia menerjemahkan panggilan API Windows menjadi [panggilan POSIX secara langsung](https://itsfoss.com/posix/), sehingga menghilangkan penalti kinerja dan memori yang biasanya terkait dengan metode lain. Dengan demikian, Wine memungkinkan integrasi aplikasi Windows secara mulus ke dalam desktop Linux.

## Instalasi Wine di Kali Linux
Untuk menginstal Wine di Kali Linux, pengguna dapat menggunakan perintah berikut di terminal:

```bash
sudo apt install wine
```

Setelah instalasi, pengguna perlu memastikan bahwa versi terbaru dari Wine telah terpasang. Namun, untuk menjalankan aplikasi 32-bit, pengguna juga memerlukan paket tambahan yang disebut `wine32`. Sebelum menginstal `wine32`, pengguna harus menambahkan arsitektur 32-bit ke sistem dengan perintah berikut:

```bash
sudo dpkg --add-architecture i386
```

Setelah menambahkan arsitektur, sistem perlu diperbarui untuk repositori arsitektur baru dengan menjalankan:

```bash
sudo apt update
```

Kemudian, pengguna dapat menginstal `wine32` dengan perintah:

```bash
sudo apt install wine32
```

Proses ini mungkin memakan waktu beberapa menit tergantung pada kecepatan internet dan kinerja sistem. Setelah instalasi selesai, pengguna akan diminta untuk me-restart beberapa pustaka; pilih "ya" untuk melanjutkan. Setelah itu, disarankan untuk merestart PC.

### Konfigurasi Wine
Setelah instalasi, pengguna perlu mengonfigurasi Wine untuk menjalankan aplikasi Windows. Ini dapat dilakukan dengan membuka konfigurasi Wine menggunakan perintah:

```bash
winecfg
```

Di dalam jendela konfigurasi, pengguna dapat mengatur berbagai opsi seperti versi Windows, Libraries, Graphics, Audio, dan Drives. Untuk saat ini, disarankan untuk membiarkan pengaturan pada konfigurasi default.

## Menjalankan Aplikasi Windows di Kali Linux
Setelah konfigurasi selesai, pengguna dapat menjalankan aplikasi dan permainan Windows dengan mengetikkan perintah berikut di terminal:

```bash
wine nama_aplikasi.exe
```

Untuk membuka file `.exe` tanpa menggunakan terminal, pengguna perlu mengatur pemuat program Wine sebagai default untuk file tersebut. Ini dapat dilakukan dengan membuat tautan simbolik menggunakan perintah:

```bash
sudo ln -s /usr/share/doc/wine/examples/wine.desktop /usr/share/applications
```

Setelah tautan dibuat, pengguna dapat mengatur tautan tersebut sebagai default untuk semua file `.exe` dengan mengklik kanan pada file tersebut dan memilih "Open With...". Kemudian, gulir ke bawah untuk menemukan "Wine Windows Program Loader".

Dengan menggunakan Wine, pengguna dapat menginstal berbagai aplikasi Windows di Kali Linux. Wine juga menciptakan sistem file yang mirip dengan Windows di direktori `/home/user/.wine`{: .filepath} yang terletak di direktori tersembunyi `.wine`{: .filepath}.

### Memasang File ISO Game Windows di Wine
Untuk memasang (mount) file ISO game Windows, ikuti langkah-langkah berikut:

#### 1. Memasang File ISO
Gunakan perintah `mount`. Pertama, buat direktori untuk mount, misalnya:

```bash
mkdir ~/iso_mount
```

Setelah itu, pasang file ISO ke direktori yang telah dibuat dengan perintah berikut:

```bash
sudo mount -o loop /path/to/your/game.iso ~/iso_mount
```

Gantilah `/path/to/your/game.iso`{: .filepath} dengan path yang sesuai dengan lokasi file ISO.

#### 2. Menjalankan Installer atau Game
Setelah file ISO terpasang, jalankan installer atau game menggunakan Wine. Sebagai contoh, jika terdapat file `setup.exe` di dalam ISO, jalankan dengan perintah:

```bash
wine ~/iso_mount/setup.exe
```

#### 3. Melepaskan Mount ISO
Setelah selesai, penting untuk melepaskan mount file ISO dengan perintah berikut:

```bash
sudo umount ~/iso_mount
```

## Winetricks
Selain Wine, terdapat alat tambahan yang disebut "Winetricks". Winetricks membantu pengguna dalam menginstal berbagai permainan Windows, aplikasi, pustaka, dan font.

Untuk menginstal Winetricks, pengguna dapat menggunakan perintah berikut:

```bash
sudo apt-get install winetricks
```

Winetricks adalah alat yang ringan dan seharusnya terinstal dalam waktu singkat. Dengan Winetricks, pengguna dapat dengan mudah menginstal aplikasi, permainan, dan berbagai komponen lainnya.

Setelah menginstal perangkat lunak tersebut, jalankan alat ini dengan menggunakan perintah berikut:

```
winetricks
```
